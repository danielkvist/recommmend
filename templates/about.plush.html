ABOUT THIS PROJECT

RECOMMMEND is a webapp build using Go, Buffalo, Bootstrap and the Spotify API.

Looking for ideas for the final project of the CS50 course I was watching a video about different APIs when I came up with the idea of building a webapp that would allow users around the world to recommend their favorite artists and receive recommendations from other users. And here we are.

This project DOES NOT needs and will not ask you for your Spotify username or login credentials. The only thing that's stored is the artist you recommend.

This project is also Open Source so you can check the code by yourself here.
